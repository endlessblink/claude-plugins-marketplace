# endlessblink Claude Code Plugins

Marketplace for Claude Code plugins by endlessblink.

## Installation

```bash
# Add this marketplace
/plugin marketplace add endlessblink/claude-plugins-marketplace

# Install plugins
/plugin install claude-dev-infrastructure@endlessblink-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [claude-dev-infrastructure](https://github.com/endlessblink/claude-dev-infrastructure) | 12 AI skills, task coordination hooks, visual Kanban dashboard |
